# Agent engine platform

The Agent Engine has two main components: the platform and [connectors](../connectors/). The platform is the subscription-based, cloud component that manages credentials and data replication.

## Log in and sign up

Log in and sign up to the Agent Engine at [app.airbyte.ai/](https://app.airbyte.ai/).
